# Daily standup po angielsku / The Daily Standup

## Grupa 1
| polski | angielski |
|---|---|
| wczoraj skończyłem / domknąłem | yesterday I wrapped up |
| Wczoraj domknąłem stronę logowania. | Yesterday I wrapped up the login page. |
| Wczoraj skończyłem review i je zmergowałem. | Yesterday I wrapped up the review and merged it. |
| Wczoraj domknąłem tickety ze sprintu. | Yesterday I wrapped up the sprint tickets. |
| Wczoraj w końcu domknąłem tego buga z cache. | Yesterday I finally wrapped up that caching bug. |
| brać się za coś / zaczynać (dziś) | to pick something up |
| Dziś biorę się za API. | Today I'm picking up the API work. |
| siedzieć z głową w czymś / być pochłoniętym | to be heads-down on something |
| (sztywno) Dzisiaj będę pracował nad testami. | Today I'll be working on the tests. |
| (luźno) Dziś siedzę z głową w testach. | Today I'm heads-down on tests. |
| nic mnie nie blokuje | nothing's blocking me |
| (sztywno) Nie mam żadnych problemów. | I don't have any problems. |
| (luźno) Na ten moment nic mnie nie blokuje. | Nothing's blocking me right now. |
| Wczoraj domknąłem stronę logowania, dziś biorę się za API, i na razie nic mnie nie blokuje. | Yesterday I wrapped up the login page, today I'm picking up the API, and nothing's blocking me so far. |

## Grupa 2
| polski | angielski |
|---|---|
| być zablokowanym przez coś | to be blocked on something |
| Jestem zablokowany przez API płatności. | I'm blocked on the payments API. |
| Utknąłem na review — czekam na nie. | I'm blocked on the review — still waiting on it. |
| Jestem zablokowany, dopóki nie dostanę dostępów. | I'm blocked until I get access. |
| Trochę utknąłem, bo czekam na deploy. | I'm a bit blocked waiting on the deploy. |
| czekać na kogoś/coś | to be waiting on someone |
| Czekam, aż Mark zmerguje swój PR. | I'm waiting on Mark to merge his PR. |
| walnąć w ścianę / utknąć na czymś | to hit a wall with something |
| Walnąłem w ścianę z tym bugiem. | I hit a wall with that bug. |
| (sztywno) Potrzebuję pomocy z konfiguracją. | I need help with the config. |
| (luźno) Przydałaby mi się pomocna dłoń przy konfiguracji. | I could use a hand with the config. |
| odblokować kogoś | to unblock someone |
| Może mnie ktoś odblokować z tymi dostępami? | Can someone unblock me on the access? |
| Utknąłem na API płatności — czekam na review Marka, więc przydałaby mi się pomocna dłoń, żeby ruszyć dalej. | I'm blocked on the payments API — I'm waiting on Mark's review, so I could use a hand to get unblocked. |

## Grupa 3
| polski | angielski |
|---|---|
| prawie gotowe / na ukończeniu | it's pretty much there |
| To już prawie gotowe. | It's pretty much there. |
| Płatności są właściwie na ukończeniu. | Payments are pretty much there. |
| Już prawie — zostały drobiazgi. | It's pretty much there, just some cleanup left. |
| Funkcja jest gotowa, dopinam testy. | The feature's pretty much there, I'm just finishing the tests. |
| wisieć na review | to be in review |
| Mój PR wisi na review. | My PR is in review. |
| wypchnąć coś do końca dnia | to have it out by EOD |
| (sztywno) Skończę to do końca dnia. | I'll finish it by the end of the day. |
| (luźno) Wypchnę to do końca dnia. | I'll have it out by EOD. |
| gotowe do merga / można wypuszczać | good to go |
| Z mojej strony można wypuszczać. | It's good to go on my end. |
| Płatności są prawie gotowe i wiszą na review — jak przejdą, wypchnę je do końca dnia. | Payments are pretty much there and in review — once they pass, I'll have them out by EOD. |

## Grupa 4
| polski | angielski |
|---|---|
| przekazać coś komuś | to hand something off |
| Przekażę to QA. | I'll hand it off to QA. |
| Przekażę resztę Ani. | I'll hand the rest off to Ania. |
| Jak skończę, przekażę to Markowi. | Once I'm done, I'll hand it off to Mark. |
| Przekażę to dalej, jak przejdzie testy. | I'll hand it off once it passes tests. |
| omówić coś poza standupem / na boku | to take something offline |
| (sztywno) Omówmy to po spotkaniu. | Let's discuss this after the meeting. |
| (luźno) Weźmy to na bok po standupie. | Let's take this offline after standup. |
| rzucić okiem na czyjś PR | to take a look at someone's PR |
| Rzucisz okiem na mój PR, jak będziesz miał chwilę? | Can you take a look at my PR when you get a sec? |
| zsynchronizować się z kimś | to sync with someone |
| Zsynchronizuję się z Anią po standupie. | I'll sync with Ania after standup. |
| wrzucić update na kanał | to post an update in the channel |
| Nie przedłużając — weźmy to na bok po standupie, zsynchronizuję się z Anią, a szczegóły wrzucę na kanał. | To keep it short — let's take this offline after standup, I'll sync with Ania, and I'll post the details in the channel. |

## Grupa 5
| polski | angielski |
|---|---|
| zajmować więcej czasu, niż się spodziewałem | to take longer than expected |
| To zajmuje więcej czasu, niż myślałem. | It's taking longer than expected. |
| Migracja idzie wolniej, niż zakładałem. | The migration's taking longer than expected. |
| Zajmuje to więcej, niż sądziłem — więcej edge case'ów. | It's taking longer than expected — more edge cases than I thought. |
| nie doszacować czegoś | to underestimate something |
| (sztywno) Źle oszacowałem ten task. | I estimated this task wrong. |
| (luźno) Trochę nie doszacowałem tego taska. | I underestimated this one a bit. |
| potrzebować jeszcze jednego dnia | to need another day |
| Będę potrzebował jeszcze jednego dnia. | I'll need another day on this. |
| uprzedzam / dla jasności | just a heads up |
| (sztywno) Informuję, że będzie małe opóźnienie. | I'd like to inform you of a slight delay. |
| (luźno) Uprzedzam — będzie mały poślizg. | Just a heads up — slight delay. |
| Uprzedzam: trochę nie doszacowałem migracji, idzie wolniej, niż zakładałem, więc będę potrzebował jeszcze jednego dnia. | Just a heads up — I underestimated the migration a bit, it's taking longer than expected, so I'll need another day. |
