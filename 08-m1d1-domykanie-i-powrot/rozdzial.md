# M1 · Dział 1: Domykanie i powrót / M1 · Part 1: Wrapping Up and Heading Home

## Grupa 1
| polski | angielski |
|---|---|
| kończyć / domykać (robotę) | to wrap up |
| Domykam robotę. | I'm wrapping up. |
| Domykam tu robotę w Norwegii. | I'm wrapping up my job here in Norway. |
| Powoli wszystko domykam. | I'm slowly wrapping everything up. |
| Domykam ostatnie sprawy przed wyjazdem. | I'm wrapping up the last few things before I leave. |
| Prawie wszystko już pozamykałem. | I've pretty much wrapped everything up. |
| Skończę za jakieś dwa tygodnie. | I'll be wrapped up in about two weeks. |
| To kiedy mniej więcej kończysz? | So when do you wrap up over there? |
| Jeszcze nie wszystko pozamykałem. | I haven't quite wrapped everything up yet. |
| (sztywno) Muszę to skończyć w tym tygodniu. | I have to finish this up this week. |
| (luźno) Muszę to ogarnąć i pozamykać w tym tygodniu. | I've got to wrap this up this week. |
| Powiedziałem Jake'owi, że domknę robotę, zanim wrócę. | I told Jake I'd wrap things up before I head back. |
| Jak już tu wszystko pozamykam, lecę do Polski. | Once I wrap everything up here, I'm flying to Poland. |

## Grupa 2
| polski | angielski |
|---|---|
| wracać (do kraju / do domu) | to head back / to head home |
| Wracam do domu. | I'm heading home. |
| Za dwa tygodnie wracam do Polski. | I'm heading back to Poland in two weeks. |
| W przyszłym miesiącu wracam do domu. | I'm heading home next month. |
| Chyba czas wracać. | I think it's time to head back. |
| Myślami wracam już od jakiegoś czasu. | I've been thinking about heading home for a while now. |
| To kiedy wracasz? | So when are you heading back? |
| Jeszcze nie wracam — najpierw muszę domknąć robotę. | I'm not heading back yet — I've got to wrap up first. |
| (sztywno) Wrócę do Polski w przyszłym miesiącu. | I will return to Poland next month. |
| (luźno) W przyszłym miesiącu zwijam się z powrotem do Polski. | I'm packing up and heading back to Poland next month. |
| Powiedziałem Jake'owi, że jak tylko domknę robotę, od razu wracam. | I told Jake that as soon as I wrap up, I'm heading straight home. |
| Tak naprawdę nie chodzi o samą przeprowadzkę — chodzi o to, że wracam na dłużej. | It's not so much the move itself — it's that I'm heading back to stay. |

## Grupa 3
| polski | angielski |
|---|---|
| na stałe / na dobre | for good |
| Wracam na stałe. | I'm heading back for good. |
| Tym razem na stałe. | This time it's for good. |
| Wyjeżdżam z Norwegii na dobre. | I'm leaving Norway for good. |
| Domykam tu wszystko na dobre. | I'm wrapping things up here for good. |
| Naprawdę wracasz na stałe? | Are you really back for good? |
| Nie wyjeżdżam znowu — tym razem zostaję na stałe. | I'm not leaving again — this time I'm here for good. |
| (sztywno) Wracam do Polski na zawsze. | I'm going back to Poland permanently. |
| (luźno) Wracam do Polski na stałe — koniec wyjazdów. | I'm back in Poland for good — no more going abroad. |
| Część mnie wciąż nie wierzy, że to na stałe. | Part of me still can't believe it's for good. |
| Minęły dwa lata, zanim zdecydowałem się wrócić na stałe. | It took me two years to decide to come back for good. |
| Powiedziałem Jake'owi, że tym razem domykam wszystko i wracam na stałe. | I told Jake that this time I'm wrapping everything up and heading back for good. |

## Grupa 4
| polski | angielski |
|---|---|
| mieć mieszane uczucia (co do) | to have mixed feelings (about) |
| Mam mieszane uczucia. | I have mixed feelings. |
| Mam mieszane uczucia co do powrotu. | I have mixed feelings about heading back. |
| Mam co do tego naprawdę mieszane uczucia. | I have really mixed feelings about it. |
| Mam mieszane uczucia co do powrotu na stałe. | I have mixed feelings about going back for good. |
| A ty nie masz mieszanych uczuć co do powrotu? | Don't you have mixed feelings about heading back? |
| Mam mieszane uczucia, odkąd zacząłem domykać robotę. | I've had mixed feelings ever since I started wrapping up. |
| (sztywno) Nie jestem pewien, co czuję w związku z wyjazdem. | I'm not sure how I feel about leaving. |
| (luźno) Sam nie wiem — mam z tym wszystkim strasznie mieszane uczucia. | Honestly, I've got really mixed feelings about the whole thing. |
| To nie tak, że nie chcę wracać — po prostu mam mieszane uczucia. | It's not that I don't want to head back — I've just got mixed feelings. |
| Powiedziałem Jake'owi wprost, że mam mieszane uczucia co do całego tego powrotu na stałe. | I told Jake straight up that I have mixed feelings about this whole heading-back-for-good thing. |

## Grupa 5
| polski | angielski |
|---|---|
| Domykam robotę i wracam do domu. | I'm wrapping up and heading home. |
| Domykam wszystko tutaj i wracam do Polski na stałe. | I'm wrapping everything up here and heading back to Poland for good. |
| Mam mieszane uczucia, ale powoli domykam sprawy. | I've got mixed feelings, but I'm slowly wrapping things up. |
| Jake pytał, kiedy domykam robotę i wracam. | Jake asked when I'm wrapping up and heading back. |
| (sztywno) Kończę pracę, wracam do kraju na zawsze i nie wiem, co czuję. | I'm finishing my job, returning home permanently, and I don't know how I feel. |
| (luźno) Domykam robotę, wracam na stałe i mam z tym totalnie mieszane uczucia. | I'm wrapping up, heading back for good, and I've got seriously mixed feelings about it. |
| Jak już tu wszystko domknę, wracam do domu na stałe — choć mam co do tego mieszane uczucia. | Once I wrap everything up here, I'm heading home for good — though I have mixed feelings about it. |
| Powiedziałem Jake'owi, że domykam robotę, wracam na stałe i, szczerze mówiąc, mam z tym mieszane uczucia. | I told Jake I'm wrapping up, heading back for good, and honestly, I've got mixed feelings about it. |
| Najdziwniejsze jest to, że domykam tu całe życie, a wciąż nie wiem, co czuję. | The strangest part is that I'm wrapping up a whole life here, and I still don't know how I feel. |
